NOTE:

All assembly commands are based on the following grammatical rule: opcode [vargs]. Auxiliary commands also follow to this rule.

The vector of arguments is passed in reverse order: A and B -> B and A

The call instruction pushes the return address and the contents of the bp register onto the stack.
If the sp register does not point to the contents of the bp register upon call, the ret instruction pushes the function result onto the top of the stack.
The function arguments are freed by the programmer (not automatically).
